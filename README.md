AI Career Counsellor – Backend
Overview

This project is the backend service for the AI Career Counsellor Web Application.
It processes user inputs related to career confusion and generates AI-based personalized career advice by analyzing the user's interest, emotional state, and decision stage.

The backend is built using Python and FastAPI, exposing REST APIs that communicate with the frontend application.

Features

AI-based career advice generation

Emotion and career interest detection

Career decision stage analysis

University recommendation suggestions

REST API architecture using FastAPI

Swagger API documentation

Cloud deployment for public access
